How to Use (Fast)

  • Decision Tool: type symptoms → press Analyze.
  • Use Tab / Shift+Tab to move between inputs/buttons.
  • Use Enter on “Analyze Condition” button when focused.
  • PDF export: Print / Save PDF or Ctrl/Cmd + P → “Save as PDF”.
  • Systems by Project: clean overview per project (less clutter).
  • Full Library: everything, searchable.
